### Setup
`cp .env.example .env`

### Local Server

`npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Testing
`npm test`

### Production Build
`npm run build`

`npm run install -g serve`
`npm run serve`
